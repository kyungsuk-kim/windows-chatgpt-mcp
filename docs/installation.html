개요

Windows ChatGPT MCP Tool은 Claude Desktop과 VS Code Claude CLI가 Windows 11 시스템에서 ChatGPT와 상호작용할 수 있게 해주는 Model Context Protocol (MCP) 서버입니다.

주요 기능

  • 🔗 원활한 통합 - MCP 프로토콜을 통해 Claude와 ChatGPT 연결
  • 🪟 Windows 네이티브 - Windows 11/10에 최적화된 강력한 자동화
  • 🛠️ 다양한 도구 - 메시지 전송, 대화 관리, 디버그 정보 제공
  • 📊 성능 모니터링 - 내장된 메트릭 및 로깅
  • 🔧 간편한 설정 - Claude Desktop 및 VS Code와 호환
  • 🛡️ 오류 처리 - 포괄적인 오류 복구 및 재시도 로직

시스템 요구사항

필수 요구사항

  • Windows 11 (권장) 또는 Windows 10 (버전 1903 이상)
  • Python 3.8 이상
  • ChatGPT 데스크톱 애플리케이션
  • Claude Desktop 또는 VS Code with Claude CLI 확장
  • 최소 4GB RAM (8GB 이상 권장)
  • 최소 500MB 여유 저장공간
  • 1920x1080 이상 해상도 (권장)

소프트웨어 요구사항

소프트웨어 버전 다운로드 링크 설치 시 주의사항
Python 3.8 이상 python.org "Add Python to PATH" 체크 필수
ChatGPT Desktop 최신 버전 공식 ChatGPT 웹사이트 로그인 상태 유지
Claude Desktop 최신 버전 Anthropic 공식 사이트 선택사항 (VS Code 대신 사용 가능)

설치 전 확인사항

설치를 시작하기 전에 다음 명령어로 Python 설치를 확인하세요:

python --version
pip --version

Python 3.8+ 및 pip 버전 정보가 표시되어야 합니다.

설치

방법 1: 자동 설치 (권장)

가장 간단하고 안전한 설치 방법입니다.

1단계: 저장소 다운로드

git clone https://github.com/example/windows-chatgpt-mcp.git
cd windows-chatgpt-mcp

2단계: 자동 설치 스크립트 실행

python scripts/install.py

설치 스크립트가 자동으로 다음 작업을 수행합니다:

  • 시스템 전제조건 확인
  • Python 의존성 설치
  • 패키지 설치
  • 설치 검증
  • 편리한 바로가기 생성

방법 2: 수동 설치

1단계: 저장소 다운로드

git clone https://github.com/example/windows-chatgpt-mcp.git
cd windows-chatgpt-mcp

2단계: 의존성 설치

pip install --upgrade pip
pip install -r requirements.txt

3단계: 패키지 설치

pip install -e .

4단계: 설치 검증

python scripts/verify_dependencies.py

방법 3: PyPI 설치 (사용 가능시)

pip install windows-chatgpt-mcp

방법 4: 독립 실행 파일

Python 설치 없이 사용할 수 있는 방법입니다.

  1. 릴리스 페이지에서 windows-chatgpt-mcp.exe 다운로드
  2. 원하는 디렉토리에 실행 파일 배치
  3. Python 설치 없이 직접 실행

설치 후 검증

1. 의존성 검증 실행

python scripts/verify_dependencies.py

다음 항목들을 확인합니다:

  • Python 버전 호환성
  • 필수 패키지 설치 상태
  • ChatGPT 애플리케이션 감지
  • Windows 자동화 권한

2. 기본 기능 테스트

python -m src.mcp_server --test

3. 패키지 가져오기 확인

python -c "import src.mcp_server; print('설치 성공!')"

일반적인 설치 문제

  • Python을 찾을 수 없음: PATH 환경변수에 Python이 추가되었는지 확인
  • 권한 거부: 관리자 권한으로 명령 프롬프트 실행
  • 패키지 설치 실패: pip 업데이트 후 개별 패키지 설치 시도
  • 바이러스 백신 차단: Python 디렉토리를 예외 목록에 추가

설정

Claude Desktop 설정

설정 파일 위치

Claude Desktop 설정 파일 위치:

%APPDATA%\Claude\mcp.json

개발 설치용 설정

소스에서 설치했거나 도구를 수정하려는 경우:

{ "mcpServers": { "windows-chatgpt": { "command": "python", "args": ["-m", "src.mcp_server"], "cwd": "C:/path/to/your/windows-chatgpt-mcp", "env": { "PYTHONPATH": "C:/path/to/your/windows-chatgpt-mcp/src", "WINDOWS_CHATGPT_MCP_LOG_LEVEL": "INFO", "WINDOWS_CHATGPT_MCP_TIMEOUT": "30" } } } }
중요: C:/path/to/your/windows-chatgpt-mcp를 실제 설치 경로로 변경하세요.

PyPI 설치용 설정

pip로 설치한 경우:

{ "mcpServers": { "windows-chatgpt": { "command": "windows-chatgpt-mcp", "args": [], "env": { "WINDOWS_CHATGPT_MCP_LOG_LEVEL": "INFO", "WINDOWS_CHATGPT_MCP_TIMEOUT": "30", "WINDOWS_CHATGPT_MCP_RETRY_COUNT": "3" } } } }

VS Code 설정

사용자 설정 (전역 설정)

파일 위치: %APPDATA%\Code\User\settings.json

{ "claude.mcpServers": { "windows-chatgpt": { "command": "python", "args": ["-m", "src.mcp_server"], "cwd": "C:/path/to/your/windows-chatgpt-mcp", "env": { "PYTHONPATH": "C:/path/to/your/windows-chatgpt-mcp/src", "WINDOWS_CHATGPT_MCP_LOG_LEVEL": "INFO" } } }, "claude.enableMcp": true, "claude.mcpTimeout": 30000 }

워크스페이스 설정 (프로젝트별 설정)

파일 위치: .vscode/settings.json

{ "claude.mcpServers": { "windows-chatgpt": { "command": "python", "args": ["-m", "src.mcp_server"], "cwd": "${workspaceFolder}/windows-chatgpt-mcp", "env": { "PYTHONPATH": "${workspaceFolder}/windows-chatgpt-mcp/src", "WINDOWS_CHATGPT_MCP_LOG_LEVEL": "DEBUG" } } }, "claude.enableMcp": true, "claude.mcpTimeout": 45000 }

환경 변수 설정

변수명 설명 기본값 가능한 값
WINDOWS_CHATGPT_MCP_LOG_LEVEL 로깅 수준 INFO DEBUG, INFO, WARNING, ERROR
WINDOWS_CHATGPT_MCP_TIMEOUT 응답 대기 시간(초) 30 10-120
WINDOWS_CHATGPT_MCP_RETRY_COUNT 재시도 횟수 3 1-10
WINDOWS_CHATGPT_MCP_WINDOW_TITLE ChatGPT 창 제목 패턴 ChatGPT 사용자 정의 정규식
WINDOWS_CHATGPT_MCP_DEBUG 디버그 모드 활성화 false true, false

일반적인 문제 및 해결책

1. "ChatGPT 창을 찾을 수 없음" 오류

증상: ChatGPT 애플리케이션을 찾을 수 없다는 오류

해결책:

  • ChatGPT가 실행 중이고 보이는지 확인
  • 창 제목이 예상 패턴과 일치하는지 확인
  • ChatGPT 창을 클릭하여 포커스 맞추기
  • 시스템 트레이에 최소화되지 않았는지 확인

2. "응답 대기 시간 초과" 오류

증상: ChatGPT 응답을 기다리다가 시간 초과

해결책:

  • 타임아웃 값 증가
  • ChatGPT가 복잡한 요청을 처리 중인지 확인
  • ChatGPT에 오류 메시지가 표시되지 않는지 확인
  • 안정적인 인터넷 연결 확인

3. "GUI 자동화 권한 거부" 오류

증상: GUI 자동화 작업에 대한 액세스 거부

해결책:

  • 관리자 권한으로 실행
  • Windows 개인정보 설정에서 앱 제어 허용
  • 바이러스 백신 실시간 보호 일시 비활성화
  • Python을 바이러스 백신 예외 목록에 추가

4. "Python을 찾을 수 없음" 오류

증상: 'python'이 내부 또는 외부 명령으로 인식되지 않음

해결책:

  • "Add Python to PATH" 체크하여 Python 재설치
  • 전체 경로 사용: C:\Python39\python.exe
  • 환경 변수 PATH에 Python 수동 추가

5. "설정 파일을 로드할 수 없음" 오류

증상: Claude Desktop 또는 VS Code에서 MCP 서버를 찾을 수 없음

해결책:

  • JSON 구문 검사: python -m json.tool %APPDATA%\Claude\mcp.json
  • 파일 경로가 올바른지 확인
  • 설정 변경 후 애플리케이션 재시작
  • 절대 경로 사용 권장

# PyInstaller 설치 pip install pyinstaller # 실행 파일 빌드 pyinstaller windows-chatgpt-mcp.spec
